랜딩페이지를 만들어줘. React + Tailwind CSS 사용.

## 페이지 컨셉
- 랜딩페이지/마케팅 대행 서비스 판매 페이지
- 타겟: 온라인 판매하는 소규모 사업자
- 톤: 직설적이고 자신감 있는 장사꾼 느낌, 허세 아닌 실력으로 증명하는 분위기

## 디자인 방향
- 다크 테마 베이스 (신뢰감 + 프리미엄)
- 강조색: 골드 또는 오렌지 계열 (CTA, 핵심 텍스트)
- 큰 타이포, 여백 넉넉하게
- 모바일 퍼스트

## 섹션 구성 (순서대로)

### 1. 히어로
- 헤드라인: "디자이너에게 맡기지 마세요. 그들은 '장사'를 모릅니다."
- 서브: "연매출 30억 찍어본 '장사꾼'이 직접 만드는 페이지는 다릅니다."
- 강조: "'팔리는 구조'를 만듭니다"
- 보장: "효과 없으면 100% 환불"
- CTA: "내 아이템 진단받고 신청하기 (월 12건 한정)"

### 2. 문제 제기
- 헤드라인: "혹시 비싼 돈 주고 만든 랜딩페이지, '예쁜 쓰레기'가 되진 않았나요?"
- 공감 포인트: 디자인은 예쁜데 매출 0원, 광고비만 태움
- 결론: "고객을 설득하는 '기획' 없이 '디자인'만 했기 때문"

### 3. 솔루션 비교
- 헤드라인: "머리 아픈 기획, 하지 마세요. 팔리는 논리는 제가 짭니다."
- 좌우 비교 레이아웃:
  - 일반 에이전시: "원고 주세요, 사진 주세요, 기획안 주세요..." (대표님이 다 해야 함)
  - 우리 방식: "제품과 장점만 던져주세요" → 기획 → 카피라이팅 → 전환 디자인
- 마무리: "대표님은 제품 소싱과 배송에만 신경 쓰세요"

### 4. 마케팅 확장
- 헤드라인: "페이지 제작이 끝이 아닙니다. 결국 '매출'이 목적 아닌가요?"
- 강조: "이커머스 마케팅 회사"
- 2개 포인트 카드:
  1. 완벽한 이해도: 기획한 사람이 소구점을 제일 잘 안다
  2. 실전 마케팅: 만든 사람이 광고 돌려야 효율 난다
- 결론: "제작부터 판매까지 끊김 없이 연결"

### 5. 증명
- 헤드라인: "말로만 전문가? 숫자로 증명합니다."
- 이미지 플레이스홀더 3개 (매출 그래프, 택배 발송량, 전환율)
- 텍스트: 10년간 수십만 개 택배, 수십억 매출
- 강조: "'설득 공식'과 '유입 공식'을 대표님 사업에 이식"

### 6. 환불 보장
- 헤드라인: "자신 없으면 돈 안 받습니다. 매출 안 나오면 100% 환불."
- 핵심: 비용이 아니라 투자
- 작은 글씨: 단순 변심 제외, 패키지 진행 시 기준 협의

### 7. 희소성
- 헤드라인: "죄송합니다. 월 12팀만 받습니다."
- 서브: "안 될 상품은 거절합니다"
- 설명: 공장식 X, 한 땀 한 땀 기획
- 강조: "계약 수락 = 성공 가능성 높다는 뜻"

### 8. FAQ + 최종 CTA
- FAQ 2개 아코디언:
  - Q: 마케팅까지 맡기면 비용 비싸지 않나요?
  - Q: 진짜 기획 안 해도 되나요?
- 긴급성: "망설이는 순간, 경쟁사는 이미 고객을 쓸어 담고 있습니다"
- 최종 CTA: "지금 내 아이템 심사 신청하기 (이번 달 3자리 남음)"
